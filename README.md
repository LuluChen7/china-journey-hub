# China Journey Hub (巴方来财)

China Journey Hub is a travel and education information platform designed to connect Brazilian users with opportunities in China, including study tours, travel programs, and collaboration with universities and organizations.

The project brand name is **巴方来财** and the website serves as an information hub for potential participants and partners.

---

# Project Overview

The website provides structured information about China-related programs and services for Brazilian audiences.

Its main goal is to present clear information and create a direct contact channel through platforms such as WhatsApp and social media.

The project follows an **iterative development approach**, meaning features and requirements will continue to evolve based on feedback.

---

# Current Development Status

The project is currently in the **Frontend Development stage**.
